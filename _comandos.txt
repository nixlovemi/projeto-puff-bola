# Docker
========
  * Sempre qual alterar algo no Dockerfile, deleta td (inclusive as imagens) e rebuilda
  * Iniciar containers: docker-compose up -d
  * Mostrar containers: docker ps
  * Remover containers: docker rm -f $(docker ps -a -q)
  * Abrir bash: docker exec -it [NOME CONTAINER] bash
